Arduino Logic Gates

Exploring fundamental digital logic through physical hardware.

This project focuses on the implementation and explanation of key digital logic gates — NOT, OR, AND, and NAND — using Arduino boards and basic electronic components.

Overview

Logic gates are essential components of modern computing systems. This collaborative project, carried out by Team #15 with four members, seeks to convert abstract Boolean logic into physical, observable actions.

Rather than addressing every possible gate, our focus was on a selected set — NOT, OR, AND, and NAND. We thoroughly examined how each gate operates, how input values affect the output, and how logical decisions are represented through electrical signals.

Team & Process

Each team member played a role in building, testing, and explaining the logic gates. By utilizing Arduino boards, breadboards, LEDs, resistors, and input switches, we demonstrated how different logical conditions lead to visible results.

A key part of the project involved explaining the behavior of each gate. We provided detailed, step-by-step explanations of each gate's function, covering topics such as truth tables, signal flow, and the practical applications of digital logic.

Implemented Gates

  • NOT
  • OR
  • AND
  • NAND
