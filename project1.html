Arduino Logic Gates

Exploring fundamental digital logic through physical hardware.

This project aims to implement and explain core digital logic gates — NOT, OR, AND, and NAND — using Arduino boards and discrete electronic components.

Overview

Logic gates are the fundamental building blocks of modern computing systems. The project, created as a team effort (Team #15), consists of four members and aims to transform abstract Boolean logic into physical, observable behaviors.

Instead of covering every possible gate, we focused on a select group — NOT, OR, AND, and NAND — and thoroughly explored how each gate functions, how inputs influence outputs, and how logical decisions are represented via electrical signals.

Team & Process

Each member contributed to the construction, testing, and explanation of the logic gates. Using Arduino boards, breadboards, LEDs, resistors, and input switches, we demonstrated how various logical conditions produce tangible outcomes.

A central aspect of the project was explaining the behavior of each gate. We provided step-by-step walkthroughs of each gate's operation, delving into truth tables, signal flow, and the real-world significance of digital logic.

Implemented Gates

  • NOT
  • OR
  • AND
  • NAND
