Arduino Logic Gates

Exploring fundamental digital logic through physical hardware.

This project focuses on implementing and explaining core digital logic gates — NOT, OR, AND, and NAND — using Arduino boards and discrete electronic components.

Overview

Logic gates form the foundation of all modern computing systems. This project was developed as a team effort (Team #15), consisting of four members, with the goal of translating abstract boolean logic into tangible, physical behavior.

Rather than implementing every possible gate, we focused on a selected subset — NOT, OR, AND, and NAND — and deeply analyzed how each gate works, how inputs affect outputs, and how logical decisions are represented through electrical signals.

Team & Process

Each team member contributed to building, testing, and explaining the logic gates. Using Arduino boards, breadboards, LEDs, resistors, and input switches, we demonstrated how different logical conditions translate into observable outcomes.

A key part of the project was explanation: we walked through each gate’s behavior step by step, discussing truth tables, signal flow, and real-world implications of digital logic.

Implemented Gates

  • NOT
  • OR
  • AND
  • NAND
